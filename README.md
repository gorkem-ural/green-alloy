# 🧪 Green Alloy AI - Mühendislik Dashboard'u

Metalurjide sürdürülebilir alaşım tasarımı ve AI destekli analiz aracı.

## 🚀 Proje Hakkında
Bu proje, **Bilecik Şeyh Edebali Üniversitesi** 1. sınıf öğrencisi olarak, metalurji derslerinde öğrendiğimiz teorik bilgileri Python ve Yapay Zeka ile birleştirmek amacıyla geliştirilmiştir.

## ✨ Öne Çıkan Özellikler
- **13 Farklı Metal Seçeneği:** Geniş metal veri seti ile çalışma imkanı.
- **Anlık Görselleştirme:** Seçilen oranların anında grafiklenmesi.
- **Gemini AI Entegrasyonu:** Alaşım özelliklerine göre sektörel analizler.
- **Mühendislik Raporu:** (Yakında) PDF olarak rapor indirme özelliği.

## 🛠️ Kurulum ve Çalıştırma
Projeyi yerelde çalıştırmak için:
1. Repoyu klonlayın: `git clone [REPO_LINKIN]`
2. Gereksinimleri yükleyin: `pip install -r requirements.txt`
3. Çalıştırın: `streamlit run main.py`
